Frequently Asked Questions

How does the CSS-only accordion work?

It uses HTML's native <details> and <summary> tags. The browser handles the open and closed states automatically without any JavaScript overhead.

Is this component fully accessible?

Yes. Because it uses semantic HTML tags, keyboard navigation (using Tab and Enter/Space) and screen readers are supported right out of the box natively.

Can I customize the color scheme easily?

Absolutely. You can modify the CSS variables located inside the :root selector at the top of the stylesheet to match your own branding colors.

How do I make an item open by default?

Simply add the boolean open attribute directly inside the opening <details> tag of the specific question you want expanded.
